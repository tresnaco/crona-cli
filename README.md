# Crona CLI.

## Installing
```bash
$ npm install crona -g
```
## Starting
To start you can login:
```bash
$ crona login
```
Or you can register:
```bash
$ crona register
```
## Tokens
To create a token use this command

```bash
$ crona token:create
```

### Listing tokens

```bash
$ crona token:list
```
